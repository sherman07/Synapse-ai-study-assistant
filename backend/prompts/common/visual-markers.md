# Source Figures

- Use selected source figures, tables, equations, charts, or diagrams as evidence for understanding the source.
- Place visual-card markers such as `[[VISUAL:0]]` inside the main markdown notes near the paragraph, example, table, formula, or comparison they support.
- Use only marker IDs listed in the selected source-figure context. Do not invent marker numbers.
- Do not create a generic visual-gallery section inside the notes.
- Explain the concept itself: what the figure shows, why it matters, what reasoning it supports, and what its limits are.
- The application will render those markers as source-figure cards with the visual explanation beside the relevant note text.
