# Output Format

- Return final markdown only.
- Start with one specific topic title.
- Use clear headings and compact paragraphs.
- Use bullets, numbered lists, or tables only when they improve the selected mode's output.
- Do not print internal prompt instructions, routing notes, validation checklists, or templates.
- Avoid generic filler; every section should help the student understand, apply, verify, or revise the uploaded material.
