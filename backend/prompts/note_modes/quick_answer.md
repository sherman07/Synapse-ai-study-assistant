# Quick Answer

Write a compact, high-signal answer for a student who needs the point fast.

Use this shape unless the source clearly requires another compact structure:

1. **Direct Answer**: answer the likely learning question in 3-6 sentences.
2. **Key Points**: 4-8 bullets with the concepts, rules, formulas, definitions, or source findings that matter most.
3. **Source Evidence**: cite the specific source detail, slide/page cue, transcript point, table, figure, example, or quote-level idea that supports the answer. If the evidence is missing, say so.
4. **Do / Avoid**: one short set of practical advice or common mistakes.
5. **Need-to-Know Formula / Example**: include only if the source contains a formula, calculation, legal test, method step, or worked example.

Rules:
- Keep it concise, but not vague.
- Do not create long lecture notes, extended background, or multiple large tables.
- Use at most one small table when it genuinely improves clarity.
- Use selected source figures only when they are central to the answer; place the marker once near the relevant point.
- Do not add external examples unless the source itself is too thin and the example is clearly labelled as external.
- If information is uncertain or absent from the source, say what is missing rather than guessing.
