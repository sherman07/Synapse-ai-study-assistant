# Detailed Explanation

Write a thorough study explanation that teaches the material in a logical order without becoming a professor-style mega pack.

Use this shape:

1. **Learning Question**: state what the source is trying to teach.
2. **Core Explanation**: teach each major idea in clear paragraphs.
3. **How the Reasoning Works**: explain the mechanism, causal chain, proof logic, legal test, method, or calculation path.
4. **Worked Examples and Evidence**: reconstruct source examples, formulas, cases, tables, figures, or transcript moments.
5. **Concept Comparison**: add a compact table when two or more ideas need comparison.
6. **Common Mistakes**: explain realistic confusions and the correction.
7. **Revision Checklist**: end with what the student should be able to define, explain, compare, calculate, or critique.

Rules:
- Prefer clarity and teaching sequence over exhaustive coverage.
- Include enough detail that the student can understand the idea without reopening the source.
- Use source figures when they genuinely help the explanation; introduce the figure before the marker and interpret it after.
- If the source is mathematical, include formulas with renderable LaTeX and show the meaning of each variable.
- If the source is research-based, teach question -> method -> result -> interpretation -> limitation.
- If the source is legal or policy material, teach rule -> element/test -> consequence -> application.
- Do not invent unsupported facts; label outside examples clearly.
